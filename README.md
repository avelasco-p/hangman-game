## Setup
### Requirements

To setup and install the games you'll need

* bash
* postgresql

### Install

``` bash
$ git clone https://github.com/avelasco-p/hangman-game
$ cd hangman-game
$ ./setup.sh database
$ ./setup.sh install

# you're done!
# to play:
$ hangman
```
